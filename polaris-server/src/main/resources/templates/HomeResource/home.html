Welcome to the POLARIS server API

This server is the central authority for the Polaris Proof-of-Location system. It manages device registration, validates cryptographic PoL tokens, and orchestrates a secure, asynchronous communication channel with remote beacons.

API Endpoints

All API endpoints are located under the /api/v1/ path.

POST/api/v1/register

Registers a new mobile device and returns an API key and initial beacon provisioning data.

GET/api/v1/beacons

Returns the list of all known beacons. Requires a valid x-api-key header.

POST/api/v1/tokens

Submits a Proof-of-Location token for validation. Requires a valid x-api-key header.

GET/api/v1/payloads

Fetches pending secure commands from the server for beacon delivery. Requires a valid x-api-key header.

POST/api/v1/payloads/ack

Submits an acknowledgment from a beacon for a delivered command. Requires a valid x-api-key header.

Admin interface

A simple web interface is available for managing beacons and creating new commands.
